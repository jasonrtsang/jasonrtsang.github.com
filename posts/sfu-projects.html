SFU: Favourite technical projects
Small RISC Processor + FGPA Sound Processing 2015
ENSC 350 – Digital System Design

There are two projects I loved from this course. The first was designing and simulating a small RISC processor on a FPGA. It ran 20 bits instructions with 16 bits data: for instance, a small FIR filter. The second utilized the sound processing capablities of the audio CODEC chip on the DEV board. Using the line-in and line-out ports, the FPGA was configured for feature adjustments such as stereo/ mono audio, volume control, equalization effects, etc. routed to the boards IOs.

Prototype Butterworth Active Filter 2015
ENSC 320 – Electric Circuits II

This is a low-pass filter that amplifies the signals from 300-3400Hz by about 3dB, and suppress all frequencies above 11000Hz by 30dB below the passband gain. A Butterworth design was the best choice for this task because of the maximally flat passband it offers. The premise of this filter was to remove high audio-frequency noise in intercom speech, that was picked up and amplified by interfering main communication signals.

Graphical UART Performance Monitor 2014
ENSC 351 – Embedded and Real Time System Software

The goal of this project was to create our own VGA and UART drivers for an embedded Linux system, which would then be used to run a graphical monitor that displays runtime performance characteristics of the UART interface. AKA - something that shows how fast you're typing. The entire project was spread across the term and felt more like development work than school work. Hence, one of my favourite team projects!

ESSS Website Redesign 2014
SFU Engineering Science Student Society (ESSS)

It was time for the ESSS to get some rebranding! Things were dated, so I took our self-hosted website and gave it a new WordPress makeover. WordPress was the logical choice as anyone could easily add and update content on the site. It's design is based off the Hoffman template, which was then heavily modified.
